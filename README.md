# 🚀 Kairos Backend Starter

A production-ready **Node.js + Express + TypeScript** backend starter template designed to speed up development and provide a clean, scalable architecture.

---

## ⚡ Features

- ⚙️ Express + TypeScript setup
- 🔐 Authentication ready (JWT + bcrypt)
- 🗄️ MongoDB integration (Mongoose)
- 🧪 Input validation with Zod
- 🛡️ Security middleware (CORS, cookie-parser)
- 📦 Clean modular architecture
- 🧹 ESLint + Prettier configured
- 🔥 Ready for production scaling

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/SadikMuktadir/backend-starter-package.git
```
### 2. Install dependencies

```bash
npm install
```
### 3. Setup environment variables

```bash
NODE_ENV=
PORT=
DATABASE_URL=
JWT_SECRET_TOKEN=
```
### 4. Run development server

```bash
npm run dev
```
### Scripts

```bash
npm run dev       # Run in development mode
npm run build     # Build TypeScript to JS
npm start         # Run production build
npm run lint      # Check code quality
npm run lint:fix  # Fix lint issues
npm run format    # Format code with Prettier
```

## 🧱 Tech Stack

This backend starter is built with modern technologies:

- ⚙️ Node.js – Runtime environment
- 🚀 Express.js – Fast web framework
- 🟦 TypeScript – Type-safe JavaScript
- 🍃 MongoDB + Mongoose – Database & ODM
- 🔐 JWT – Authentication system
- ✅ Zod – Schema validation
- 🧹 ESLint + Prettier – Code quality & formatting
